# Habit Tracker API

## Описание
Это базовый пет-проект. Бэкенд написан на Java, фронтенд нужно написать на Vanilla JS.

## Backend (Папка /backend)
Сервер поднимается локально на `http://localhost:8080`.
Текущие эндпоинты:
- `GET /habits` - Отдает массив в формате JSON. Пример: `[{"id": 1, "name": "Перестать тупить", "done": false}]`
- `POST /habits` - Принимает JSON вида `{"name": "Новая привычка"}`.

## Frontend (Папка /frontend)
ИНСТРУКЦИЯ ДЛЯ АГЕНТА JULES:
1. Создай файлы `index.html`, `style.css` и `app.js` строго внутри директории `/frontend`.
2. Напиши логику на Vanilla JS. Сделай fetch-запросы (GET для получения списка и POST для добавления) к эндпоинтам `http://localhost:8080`.
3. Отрисуй минималистичный и рабочий UI списка привычек. Не используй React, Vue или другие фреймворки. Только чистый JS, HTML и CSS.
